Програмне забезпечення: стилі CSS для документації

Головна | Архітектура ПЗ | Приклади

Вступ

Опис програмних продуктів часто публікують у вигляді веб-документів. Зовнішній файл css/styles.css задає вигляд для body, заголовків та посилань на розділи про системне та прикладне ПЗ.

Цей абзац оформлений внутрішнім стилем: наголос на важливості оновлень безпеки та ліцензійних обмежень при розгортанні ПЗ.

Заголовок використовує #main-heading з !important у зовнішньому CSS для демонстрації пріоритету правил.
