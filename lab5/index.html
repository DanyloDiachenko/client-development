ПЗ: швидкий доступ

Програмне забезпечення: блоки, float, псевдокласи

Головна Документація Макет

Посилання стилізовані для навігації по розділах про системне та прикладне ПЗ (:link, :visited, :hover, :active).

Блочні та рядкові елементи

Блок <div> — окремий рядок для опису модуля ядра або сервісу.
Другий блок — наприклад, панель інструментів IDE.

Рядковий <span> для термінів: API, SDK, runtime.

Позиціонування та float

float: left
Панель файлів проєкту.
float: right
Властивості збірки (build).

Текст обтікає блоки інтерфейсу документації ПЗ. Контейнер використовує clearfix після плаваючих елементів.

position: relative — зміщений блок «Примітка про сумісність версій».
position: absolute — мітка «Beta» у куті картки релізу.
position: static — звичний потік для основного тексту про ліцензію.

Псевдоклас та псевдоелементи

  • Перший пункт — етапи SDLC: аналіз, проєктування, реалізація.
  • Другий пункт — тестування та впровадження.
  • Третій пункт — супровід і виведення з експлуатації.

Програмне забезпечення — це інструмент думки, який перетворює ідеї на дії машини.
